# APK v1.6.0 Ready!

The new version of the app has been successfully built.

**File:** `rks_v1_6.apk`
**Location:** Root directory of the project

## Changes in this version:
- **Raw Material Stock Fixes:**
    - Fixed delete and edit functionality for stock items.
    - Resolved issues with custom stock item names.
- **Expenses & Usage Improvements:**
    - Added "Date View" filter to Expenses List.
    - Added "Month View" and "Date View" filters to Raw Material Usage.
    - Improved Usage List display with formatted date and cost.
- **Stats Module Overhaul:**
    - Removed top summary cards.
    - Updated "Item-wise Sales" to show Sales (kg) and Sales (Rs).
    - Updated "Customer-wise Sales" to show Sales (kg) and Sales (Rs).
    - Fixed "Customer Purchase Breakdown" to correctly display data for selected customers.
- **Gestures:**
    - Added **Pull-to-Refresh** gesture.
    - Added **Swipe-Back** gesture.

## How to Install:
1. Download the `rks_v1_6.apk` file to your Android device.
2. Open the file and tap "Install".
3. If prompted, allow installation from unknown sources.
